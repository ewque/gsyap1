# gsyap1
Питон. Функции. Двоичные файлы и коллекции

Данный код представляет собой программу, которая принимает от пользователя числа и вычисляет определенные операции над этими числами в соответствии с выбранным пользователем вариантом.

В функциях square, cube, square_roots и logarithm производятся вычисления для каждого числа из списка numbers в соответствии с выбранным вариантом. Каждая функция возвращает соответствующий список значений.

Функция calculate_option определяет выбранный вариант и вызывает соответствующую функцию для вычисления списка значений.

Функция enter_values запрашивает у пользователя числа и возвращает значения end, start и step в зависимости от введенных значений.

Функция calculator принимает список чисел и предлагает пользователю выбор типа списка для вычисления. В цикле пользователь выбирает соответствующий номер списка, после чего вызывается функция calculate_option и результат выводится на экран.

Главная функция main представляет собой основной цикл программы, в котором пользователь может вводить числа и выбирать тип списка для вычисления. В зависимости от выбора пользователя, выводятся соответствующие результаты. После завершения вычислений предлагается возможность повторить ввод чисел.

Остальные строки кода являются вспомогательными для обработки введенных значений и управления программой.
